# Scripts - Setup

This directory contains scripts used for setting up the development environment for the Country Currency App.

## Files

- `setup.sh` - Main setup script for initializing the project
- `configure_databricks_cli.sh` - Script for configuring Databricks CLI authentication

## Usage

### Initial Setup

```bash
./setup.sh
```

### Configure Databricks CLI

```bash
./configure_databricks_cli.sh
```
