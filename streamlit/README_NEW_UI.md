# Country Currency App - Новый пользовательский интерфейс

## Обзор

Это новый пользовательский интерфейс для приложения Country Currency, разработанный с современным дизайном в темной теме. Приложение обеспечивает удобный просмотр, управление и анализ данных о странах и валютах из таблицы в Unity Catalog.

## Основные возможности

### 1. Просмотр данных
- Интерактивная таблица с пагинацией
- Автоматическое обновление при изменении данных
- Отображение схемы таблицы и статистики
- Показ общего количества записей и размера данных

### 2. Фильтрация и сортировка
- Поиск по тексту во всех колонках
- Фильтрация по диапазону для числовых полей
- Выбор из выпадающих списков для категориальных данных
- Сортировка по любой колонке в обоих направлениях

### 3. CRUD операции
- Добавление записей через удобную форму с валидацией типов
- Пакетная загрузка через CSV/Excel файлы
- Редактирование данных с удобным интерфейсом
- Удаление записей с подтверждением

### 4. Визуализация данных
- Разнообразные графики (гистограммы, круговые диаграммы, линейные графики)
- Статистические сводки (среднее, медиана, стандартное отклонение)
- Настройка визуализаций (выбор осей, цветовой схемы и др.)

## Технические детали

### Архитектура приложения

Новый пользовательский интерфейс имеет модульную структуру:

- `app_new.py` - Основной файл приложения
- `ui/` - Модули пользовательского интерфейса:
  - `data_display.py` - Отображение данных и пагинация
  - `filtering.py` - Фильтрация и сортировка
  - `crud_operations.py` - Операции с данными (CRUD)
  - `visualizations.py` - Визуализация данных
  - `main_view_new.py` - Основное представление с вкладками
  - `sidebar.py` - Боковая панель навигации
  - `style.css` - CSS стили для темной темы

- `operations/data_operations_new.py` - Расширенные операции с данными
- `templates/html_components.py` - Шаблоны HTML для UI компонентов

### Требования

- Python 3.8+
- Streamlit 1.18+
- Plotly/Altair для визуализации
- Pandas для обработки данных
